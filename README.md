# Recipe Sharing

## Techstack

Framework: **React.js**  
State Management: **Zustand**
Testing: **Jest + React-testing-library**
CI/CD: **Jenkins**  
Hosting: **Netlify**  
Styling: **Tailwind CSS**  
CMS: **Contentful**
Tooling: **Vite, Eslint, Prettier, Husky, Commitlint**
